Running Electrical Wires on Top of Roof Trusses

As we were working on our house today, we noticed that we needed to add a three-way switch in the living room for the lights.  To run the wires the obvious choice was to go up and through the roof trusses.

But you can't simply bore holes through roof trusses--they're engineered.  That's especially true for ours given that they're 2x4 roof trusses.  I didn't feel good about boring holes through them even if our engineer would approve it.  

So the question became, "how do we run these wires?"  There are a few different ways to do it safely and in a code compliant manner for a non-inhabited area. 

If the area that you need to take the wire to is parallel with the start and end of the run, that's easy.  You just staple the wire to the side of the truss.  If not, it's a bit more complicated.

If you need to run the wire perpendicular to the run of the trusses, you might need to protect the wire.  From what I read of the code (though check for yourself obviously), you only need to protect the wiring once you get beyond six feet away from the entrance to the attic.  Personally, I would protect it regardless given that you're going to be up there moving lights around, messing with insulation, and who knows what else later.  

One way to protect it is to run boards on each side of the wire so that you can't step on it.  I would suggest 2x4s given that they can take the full weight of a person wheres a one-by can't always do that when the trusses are on 24" centers (very common).  

Another way to protect the wire is to run the wire up the truss and then staple the wire to the underside of the top chord of the truss (basically you're running it on the attic ceiling).  That way you'll never step on it.  

Lastly, you can run the wires close to the V where the bottom and top truss chords meet where the truss sits on the wall (one end).  However, you don't want to run the wires directly on top of the wall that the truss sits on given that you'll undoubtedly want to drill through the top plate of that wall to run wires through it.  Bring it back to the interior 6 inches to a foot, which should be enough to protect it from someone stepping on top of it.  

There might be another way if you're on the second story or if you built over a crawlspace or have floor joists on your first story.  In our case, we have floor joists and the easiest way to protect the wire was to run the wire down into the floor joists and then bring it back up in the other location.  This will likely work in most, but not all, situations.
